# Portfolio Site

A personal portfolio and resume website built with SvelteKit, Svelte 5, and Tailwind CSS. Deployed on Netlify.

## Tech Stack

- **Framework:** SvelteKit + Svelte 5
- **Styling:** Tailwind CSS (with typography and forms plugins)
- **Animations:** GSAP
- **Deployment:** Netlify

## Development

```sh
npm install
npm run dev
```

## Building

```sh
npm run build
npm run preview
```
